#Portfolio, predstavitev kraja, ... (3+ podstrani)
 ---
Izdelajte lastno predstavitveno spletno stran (portfolio) ali predstavitev svojega kraja.

Stran mora imeti najmanj tri podstrani, ki so med seboj povezane z navigacijo.
Uporabite lahko tudi že obstoječo predlogo s spleta (iščite free css, html templates), vendar jo morate ustrezno predelati. Pomembno je, da razumete in obvladate uporabljeno kodo.

Pred začetkom si načrtujte osnutek strani (vsebino in strukturo).
Pri izdelavi uporabite HTML in CSS.

Stran naj bo smiselno in estetsko oblikovana (barve, slike, ikone). Oblikovanje je prepuščeno vaši izbiri, vendar upoštevajte osnovna načela oblikovanja, predvsem barvno usklajenost.

Pri delu bodite iznajdljivi, kreativni in samostojni.
